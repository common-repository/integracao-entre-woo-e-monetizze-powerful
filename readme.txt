Integração entre Monetizze e Woocommerce por meio de Webhook

== Description ==

Este plugin não é oficial e não possui qualquer vinculo com a Monetizze ou Woocommerce. Apenas foi feito com integrações disponibilizadas por ambos.
Este plugin permite pegar os pedidos feito pela plataforma Monetizze e converte-los em pedidos no seu woocommerce através de webhook.

== HOW IT WORKS ==

Este plugin utiliza a API oficial da Monetizze para poder fazer a integração. Se os seus produtos estiverem com o mesmo nome das duas plataformas, eles serão importados nos pedidos automaticamente.

== Standard Features ==

- Configuração simples, basta cadastrar os dados de acesso que a monetizze fornece;
- Veja a lista de pedidos feitos na monetizze prontos para importar;
- Ative a opção webhook cadastrando no painel da monetizze, a url fornecida pelo plugin para importar pedidos automaticamente;
- Em qualquer um dos modos, o cliente é cadastrado no Woocommerce caso já não tenha um registro;
- *Novo!* Área de opções, escolha se deseja desativar o cliente caso o pedido seja cancelado ou reembolsado

== Installation ==

1. Upload `integracao-monetizze-wc-powers` para a pasta `/wp-content/plugins/` 
2. Ative o plugin na área de "Plugins" do Wordpress
3. No painel do Monetize cadastre a URL de Postback apresentada pelo plugin.
